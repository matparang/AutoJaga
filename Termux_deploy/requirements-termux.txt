# ============================================
# JagaChatbot — Termux-safe requirements
# Android aarch64, Python 3.11+, no Rust deps
# ============================================
#
# INSTALL ORDER MATTERS. Run these FIRST before pip install:
#
#   pkg install python-aiohttp
#   pkg install python-cryptography
#   pkg install python-tokenizers
#
# Then install litellm WITHOUT deps (to skip tiktoken/fastuuid):
#   pip install "litellm>=1.40.0,<1.76.1" --no-deps
#
# Then install this file (all pure-Python or pre-built aarch64 wheels):
#   pip install -r requirements-termux.txt
#
