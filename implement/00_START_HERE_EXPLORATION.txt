KEY CODE SNIPPETS
═════════════════════════════════════════════════════════════════════════════

Register & Execute a Tool:
  from jagabot.agent.tools import ToolRegistry
  registry = ToolRegistry()
  registry.register(tool_instance)
  result = await registry.execute("tool_name", {"param": "value"})

Connect MCP Server:
  from nanobot.agent.tools.mcp import connect_mcp_servers
  from contextlib import AsyncExitStack
  async with AsyncExitStack() as stack:
      await connect_mcp_servers(mcp_config, registry, stack)

Load All Jagabot Tools:
  from jagabot.guardian.tools import register_jagabot_tools
  register_jagabot_tools(registry)

Evolution Cycle:
  from jagabot.evolution.engine import EvolutionEngine
  engine = EvolutionEngine()
  result = engine.cycle()

Load Configuration:
  from jagabot.config import load_config
  config = load_config()
  api_key = config.get_api_key("deepseek")

(More examples in QUICK_REFERENCE.md)
